# CalendarEvents
A simple transparent background KWGT widget that shows upcoming calendar events.

# How To
1. Install KWGT app from play store.
2. Download the "Celender Events.kwgt" file and import it in kwgt app. Position the widget as you want using settings inside Stack group, position tab.
